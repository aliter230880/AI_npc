# 🤖 AI NPC Platform

Интерактивные персонажи с искусственным интеллектом.

## Быстрый старт

```bash
npm install
cp gpt5.3-ai-npc-platform/backend/.env.example gpt5.3-ai-npc-platform/backend/.env
# Отредактируйте .env и добавьте OPENAI_API_KEY
npm run dev:all
```

## Ссылки

- API: http://localhost:8787/health
- Web: http://localhost:5173

## Обновление через GitHub Actions

1. Создайте Issue с меткой "update"
2. Или запустите Actions вручную
3. Скачайте обновлённые файлы
